# dames.koplugin

A Draughts (Checkers) plugin for [KOReader](https://github.com/koreader/koreader).

## Rules

Two players alternate moving pieces diagonally on dark squares. Jump over opponent pieces to capture them; multiple jumps in one turn are mandatory if available. A piece reaching the far end becomes a King and can move/capture in both directions. Capture all opponent pieces or leave them unable to move to win.

## Features

- **Two-player local mode**
- **Mandatory capture** — jumps are forced when available
- **King promotion** — pieces crown automatically
- **Move highlight** — shows valid moves for selected piece
- **Undo** — take back the last move
- **Auto-save** — game state saved and restored on next launch

## Installation

1. Download `dames.koplugin.zip` from the [latest release](../../releases/latest).
2. Extract into the `plugins/` folder of your KOReader data directory.
3. Restart KOReader.
4. Open the menu → **Tools** → **Draughts**.

## Controls

| Action | How |
|--------|-----|
| Select a piece | Tap it |
| Move / capture | Tap destination |
| Undo last move | Tap **Undo** |
| New game | Tap **New** |
| Show rules | Tap **Rules** |
